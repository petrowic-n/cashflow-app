# 💸 BudgetFlow

A personal budget tracker built with vanilla JavaScript using OOP architecture. Built as a learning project to practice class inheritance, encapsulation, DOM manipulation, and localStorage persistence.

## 🚀 Features

- Live balance, stats, and category breakdown updated on every change
- Delete transactions
- Add income and expense transactions
